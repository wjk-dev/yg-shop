# ZZ-Shop
**项目简介：**
该商城是基于原生微信小程序来实现的。
**已完成功能：**
- 登录授权
- 商品添加进购物车、购物车删除商品
- 展示个人中心、首页、商品详情
- 搜索商品
- 收藏商品
- 订单查询

接口在此：https://blog.csdn.net/weixin_43950643/article/details/107796815
